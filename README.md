# 宠物领养管理系统简介 java12

## 项目概述

宠物领养管理系统是一个基于JavaWeb的Web应用程序，旨在提供宠物领养相关的信息管理和操作平台。本系统通过前后端分离的方式，实现了用户与管理员两端的功能需求。

## 技术栈

- 前端：JSP
- 后端：Servlet
- 数据库：MySQL
- 服务器：Tomcat
- 开发环境：IDEA、MySQL、JDK、Tomcat

## 功能模块

### 用户端

- 登录注册功能
- 查看宠物知识
- 查看领养中心
- 宠物领养操作
- 查看团队博客
- 查看团队展示
- 提交领养申请

### 管理员端

- 宠物信息管理
- 用户信息管理
- 管理员信息管理
- 领养信息管理
- 评论信息管理
- 志愿者信息管理

## 系统角色

- 普通用户：登录注册，浏览信息，领养宠物等
- 管理员：系统管理，信息维护，用户管理等

## 运行环境

- 服务器：Tomcat 7+
- 数据库：MySQL 5.5+
- JDK版本：JDK 1.8+

## 部署步骤

1. 配置好相关运行环境（如MySQL、JDK、Tomcat等）
2. 导入数据库脚本
3. 部署应用到Tomcat服务器
4. 访问对应地址即可使用
